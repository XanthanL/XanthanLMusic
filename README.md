# XanthanL | Electric Mirage

Official website for independent artist XanthanL. Featuring the debut album "Electric Mirage".

## Tech Stack

- **Framework**: React 19 (TypeScript)
- **Bundler**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Background**: Custom Canvas Particle System (Starfield)

## Development

```bash
cd app
npm install
npm run dev
```

## Production

```bash
npm run build
```
